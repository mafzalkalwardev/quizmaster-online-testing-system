STEP 3: Install dependencies
  $ npm install
  
  This will install:
  - express (web framework)
  - mongoose (database ORM)

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: MongoDB connection error
  → Check MongoDB is running on port 27017
  → Verify connection string in config/db.js
  → Check database name: quizmasterDB

Issue: Port 3000 already in use
  → Change PORT in app.js or kill process using port 3000

Issue: Views not rendering
  → Verify views path in app.js
  → Check EJS engine is configured
  → Ensure .ejs files are in views/ directory

Issue: Styles not loading
  → Check static files path in app.js
  → Verify CSS file at public/css/style.css
  → Check browser console for errors

Issue: Session not working
  → Ensure express-session is installed
  → Check session configuration in app.js
  → Clear browser cookies and try again

Issue: Dependencies not installing
  → Delete node_modules and package-lock.json
  → Run npm install again
  → Check Node.js version (v14+)
