Welcome! Your complete QuizMaster project is ready. Follow these steps to get started.


⚡ INSTANT START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

Step 1: Open Terminal/PowerShell
$ cd C:\Users\pc\Desktop\backend_project\quizmaster

Step 2: Install dependencies
$ npm install
(This takes 1-2 minutes)

Step 3: Ensure MongoDB is running
Start MongoDB service or run:
$ mongod

Step 4: Start the application
$ npm run dev

Step 5: Open browser
Go to: http://localhost:3000

Done! ✅


📋 FIRST TIME SETUP DETAILS
═══════════════════════════════════════════════════════════════════════════════

PREREQUISITES:
✓ Node.js v14+ (check: node --version)
✓ npm (check: npm --version)
✓ MongoDB (check: mongo --version)

INSTALLATION:

1. Navigate to project:
   cd quizmaster

2. Install all dependencies:
   npm install
   
   This installs:
   • express (web framework)
   • mongoose (database ORM)
   • ejs (template engine)
   • express-session (sessions)
   • nodemon (auto-reload)

3. Verify MongoDB:
   
   Windows:
   - MongoDB should be running as a service
   - Or open CMD and run: mongod
   
   macOS/Linux:
   - brew services start mongodb-community
   - Or: mongod

4. Start application:
   
   Development (recommended):
   npm run dev
   
   This shows:
   ✓ Server running at http://localhost:3000
   ✓ MongoDB connected
   ✓ Ready for requests

5. Access application:
   Open browser: http://localhost:3000


🎮 TESTING THE APPLICATION
═══════════════════════════════════════════════════════════════════════════════

STUDENT TEST WORKFLOW:

1. Register as Student:
   • Click "Register" on home page
   • Fill in name, email, password
   • Submit
   • Login with your credentials

2. Browse Quizzes:
   • Go to "Quizzes" section
   • See all available quizzes
   • Click quiz title for details

3. Attempt Quiz:
   • Click "Start Quiz"
   • Answer all MCQs
   • Note the timer
   • Click "Submit Quiz" when done

4. View Result:
   • See score and percentage
   • Check performance status
   • Go back to browse more quizzes

5. Check Progress:
   • Go to "My Results"
   • View all your quiz attempts
   • See scores and dates

6. See Rankings:
   • Go to "Leaderboard"
   • See top performing students
   • Find your rank


ADMIN TEST WORKFLOW:

1. Login as Admin:
   • Go to Login page
   • Select "admin" role
   • Create admin account or login

2. Admin Dashboard:
   • See total stats
   • View quick actions
   • Check top quiz

3. Create Quiz:
   • Click "Create New Quiz"
   • Fill quiz details:
     - Title
     - Category
     - Difficulty
     - Total marks
     - Time limit
   • Submit

4. Add Questions:
   • Click "Add Questions"
   • Fill question details:
     - Question text
     - Options (A, B, C, D)
     - Correct answer
     - Marks
   • Add multiple questions

5. View Analytics:
   • Go to "Analytics"
   • See attempts per quiz
   • View top students
   • Check score distribution

6. Manage Students:
   • View all registered students
   • See registration dates
   • Track participation

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: Cannot connect to MongoDB
Solution:
✓ Ensure MongoDB is running
✓ Check connection string in config/db.js
✓ Port 27017 should be open

Problem: Port 3000 already in use
Solution:
✓ Kill process using port 3000
✓ Or change PORT in app.js

Problem: Dependencies won't install
Solution:
✓ Delete node_modules folder
✓ Delete package-lock.json
✓ Run npm install again

Problem: Views not rendering
Solution:
✓ Check views folder exists
✓ Verify .ejs files present
✓ Restart server

Problem: Styles not loading
Solution:
✓ Check public/css/style.css exists
✓ Verify static route in app.js
✓ Clear browser cache
