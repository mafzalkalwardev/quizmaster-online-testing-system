DATABASE SCHEMAS
================

User Schema:
{
  name: String,
  email: String (unique),
  password: String,
  role: String (admin/student),
  createdAt: Date
}

Quiz Schema:
{
  title: String,
  category: String,
  difficulty: String,
  totalMarks: Number,
  timeLimit: Number,
  createdBy: ObjectId,
  createdAt: Date
}

Question Schema:
{
  quizId: ObjectId,
  questionText: String,
  optionA: String,
  optionB: String,
  optionC: String,
  optionD: String,
  correctAnswer: String,
  marks: Number
}

Attempt Schema:
{
  studentId: ObjectId,
  quizId: ObjectId,
  answers: Map,
  score: Number,
  totalQuestions: Number,
  attemptedAt: Date
}

Result Schema:
{
  studentName: String,
  quizTitle: String,
  score: Number,
  totalMarks: Number,
  percentage: Number,
  submittedAt: Date,
  studentId: ObjectId,
  quizId: ObjectId
}


API ENDPOINTS CREATED
====================

Authentication:
GET    /register
POST   /register
GET    /login
POST   /login
GET    /logout

Quizzes (RESTful):
GET    /quizzes                      - List all quizzes
GET    /quizzes/:id                  - Get quiz details
GET    /quizzes/admin/create         - Create page
POST   /quizzes/admin/create         - Create quiz
GET    /quizzes/admin/:id/edit       - Edit page
PUT    /quizzes/admin/:id/edit       - Update quiz
DELETE /quizzes/admin/:id/delete     - Delete quiz

Questions:
GET    /questions/quiz/:quizId/questions         - List questions
GET    /questions/quiz/:quizId/add               - Add form
POST   /questions/quiz/:quizId/add               - Create
GET    /questions/quiz/:quizId/:id/edit          - Edit form
PUT    /questions/quiz/:quizId/:id/edit          - Update
DELETE /questions/quiz/:quizId/:id/delete        - Delete

Quiz Attempts:
GET    /quiz/:quizId/start          - Start quiz
POST   /quiz/:quizId/submit         - Submit quiz
GET    /result/:resultId            - View result
GET    /my-attempts                 - Student attempts
GET    /my-results                  - Student results

Admin:
GET    /admin/dashboard             - Admin dashboard
GET    /admin/quizzes               - Manage quizzes
GET    /admin/students              - Manage students
GET    /admin/leaderboard           - Leaderboard
GET    /admin/analytics             - Analytics
GET    /admin/quiz/:id/attempts     - Quiz attempts


INSTALLATION & RUNNING
======================

Prerequisites:
✓ Node.js (v14+)

Installation Steps:

2. npm install

Access Application:
